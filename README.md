# JogoDaVelha# Jogo da Velha em C#

Este é um simples jogo da velha implementado em C# para dois jogadores ou para competir contra uma máquina. 
Você pode jogar contra um oponente básico ou um oponente especializado (expert) que toma decisões mais inteligentes. Divirta-se!

## Como Jogar

### 1. Clone este repositório:

   ```bash
   git clone https://github.com/diegoluanfs/JogoDaVelha
   ```

### 2. Navegue até o diretório do projeto:

   ```bash
   cd JogoDaVelha
   ```

### 3. Compile o projeto:

   ```bash
   dotnet build
   ```

### 4. Execute o jogo

   ```bash
   dotnet run
   ```

Siga as instruções no console para escolher as opções de jogo e aproveite a competição!

## Funcionalidades
Jogue contra um amigo no modo de dois jogadores.
Desafie a máquina em dois níveis de dificuldade: básico e expert.
Interface de console simples e intuitiva para uma experiência de jogo sem complicações.
Recomece o jogo sempre que desejar.
